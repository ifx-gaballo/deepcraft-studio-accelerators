# Live Data Collection

## Overview

This starter project shows you how to collect and label image data live. This can be done directly from your PC. 

The graph that you see in the Main.imunit contains Image collection unit and predefined labels unit boxes.

By following this example, you will learn how to collect data and annotate it live, which will drastically reduce the time spent on collecting data.

## Concepts 

By opening the Main.imunit file you will see the graph which constitutes this starter project.

In this graph there are image collection unit and predefined labels unit boxes

To run this graph , double click the image collection unit box.

After capturing, you can check images save it to disk, to be used for later model training or evaluation.

## Trying it out

1. Open the Main.imunit file from the Solution Explorer.
2. Define your Predefined Label list.
3. Click Image Collection unit and choose camera and data folder configurations.
4. Double click the Image Collection unit.
5. Wait for the session to open.
6. Press the capture button.
7. After capturing images select the wanted label.
8. After editing labels save changes.
